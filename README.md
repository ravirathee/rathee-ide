# Rathee IDE - Best Competitive Coding IDE

A dependency-free local browser IDE for competitive programming. It runs Python 3 and C++20, stores stdin in `workspace/input.txt`, and writes stdout to `workspace/output.txt`.

## Start

```bash
npm run dev
```

Open:

```text
http://127.0.0.1:4173
```

## Features

- Run Python and C++ code from the editor.
- Debug mode with Python tracebacks, C++ compiler diagnostics, sanitizer-enabled C++ builds, runtime stderr, and timeout reports.
- Competitive-programming I/O: editor input becomes `input.txt`; program stdout becomes `output.txt`.
- Vertical and horizontal split layouts.
- No npm dependencies required.
